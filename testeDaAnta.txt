ESTA TUDO FLUINDO BEM LEK TO NEM ACREDITANDO
======================================================================
1-Criar uma pasta do projeto
2-Entrar na pasta para iniciar o repositorio com o comando "git init", dentro da pasta de repositorio
3-Após criar o projeto ja com sua base mandar um "git status", nesse comando ele ira me trornar a branch que eu estoi, se eu ja fiz algum commit e os arquivos não rastreados
4-  É um "git add.\nome do arquivo" que nao foi rastreados
5- "git status" para ver se esta tudo ok
6-"git commit -m "msg" " para criar a mesnsagem que sera anexada junto do commit 
7-"git remote add origin + url do repositorio" com esse comando é adicionado um repositorio ja existente no git hub
8-"git push -u origin master" executar esse comando na primeira vez que for dar um push para que o repositorio local saiba aonde ele vai parar.
